Cómo usar el material

  • Presentaciones: navegá con ← / →, O para vista general y F para pantalla completa.
  • Laboratorios: subí el template.yaml a CloudFormation (o usá aws cloudformation deploy). Son de bajo costo y borrables; cada guía cierra con la limpieza.
  • Offline: también podés clonar el repo y abrir los HTML localmente. Para evitar restricciones de file://, serví la carpeta con python3 -m http.server.
